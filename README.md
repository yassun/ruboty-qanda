# Ruboty::Qanda

ruboty plugin for knowledge Q&A API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-qanda'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-qanda

## Usage

```
> ruboty tell me 人類で最初の宇宙飛行士は？
インターネットで調べたところ、一位は、ガガーリンです。
```
